# casandrea-instagram-media

Repo **public** servant uniquement a heberger les medias (photos / videos) que
l'automatisation `casandrea-instagram` publie sur Instagram.

L'API Instagram exige une URL publique pour chaque media. On utilise donc :
`https://raw.githubusercontent.com/sacksickmichael-art/casandrea-instagram-media/main/<fichier>`

## Convention

- Depose les fichiers a la racine (ou dans des sous-dossiers, ex. `2026-06/`).
- Le nom de fichier mis dans `content/queue.yml` (repo casandrea-instagram) doit
  correspondre exactement (ex. `hero-pool-golden-hour.jpg`).
- Formats : JPG/PNG pour les images, MP4 (H.264 / AAC) pour reels et stories.

> Repo public par necessite technique. N'y mets que des medias destines a etre publies.
